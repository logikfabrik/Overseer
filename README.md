# Overseer [![Build status](https://ci.appveyor.com/api/projects/status/5qb10vsch42wx2y4?svg=true)](https://ci.appveyor.com/project/logikfabrik/overseer)

Overseer is a Windows tray app for build notifications, in its early stages. It features support for AppVeyor and Visual Studio Team Services through access tokens.

## Contributions
Overseer is Open Source (MIT), and you’re welcome to contribute!

If you have a bug report, feature request, or suggestion, please open a new issue. To submit a patch, please send a pull request.
